home Building a Simple Reading Page

The main goal of this page is to make content easy to read. I used semantic HTML and simple CSS spacing rules to keep everything clear.

I also included different content types like lists, a table, media, and one small interaction to practice UI basics in one place.

Good UI should support the content instead of distracting from it.
